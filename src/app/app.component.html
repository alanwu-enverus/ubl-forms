How to use

1. Select a document type
Choose any of the 80+ UBL document types from the dropdown. The form defaults to Invoice.

2. Create or load
Create — opens an empty form showing only required fields.
Load Sample — pre-populates the form with a real UBL sample document.

3. Fill in fields
Required fields appear immediately. Use the icons below to reveal optional fields, load nested references, or add/remove array items.

4. Export JSON
The JSON panel on the right updates in real time and shows the cleaned output (empty fields are omitted automatically).

Icon guide
  • Expand optional fields for this section
  • Collapse the section
  • Load a nested reference (lazy-loaded)
  • Add a new item to the list
  • Remove this item from the list
